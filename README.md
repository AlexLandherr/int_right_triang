# Install & Use
First clone the repo:
```
git clone https://github.com/AlexLandherr/int_right_triang.git
```
Change directory to repo:
```
cd int_right_triang/
```
Create a directory called `obj`:
```
mkdir obj
```
Then build & compile using `make`:
```
make
```

Run:
```
./int_right_triang
```
